# CoPipes

This is a test version. To try it you can create a GitHub codespace.

1. Select **<>Code** -> **Codespaces** tab -> **Create codespace on main** 
2. It will start installing. You Have to wait for 2 mins in the first time. After that it will take 2/3 seconds to open up. This will open a VS Code page in your browser.
3. Open a terminal and run

```bash
./start.sh up
```

4. When it completes creating the infrastructure it shows the link to access the application. Just ctrl + click the displayed link for **CoPipes**.
